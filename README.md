# Gerador de Dicas
Projeto feito para a geração de dicas de estudo para o usuario.

## Funcionalidades
* Botão para gerar uma dica
* Dicas aleatorias

---

## Tecnologias Ultilizadas
* HTML
* CSS
* JavaScript
* VS Code
* Git e GitHub

---

## Como Usar
* 1. Baixe o repositório (Clique no Botão “Code” > Download ZIP)
* 2. Extraia a pasta
* 3. Abra o arquivo index.html no navegador

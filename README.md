# Gerador de Dicas
Projeto feito para a geração de dicas de estudo para o usuario.

## Funcionalidades
* Botão para gerar uma dica
* Dicas aleatorias

---

## Tecnologias Ultilizadas
* HTML
* CSS
* JavaScript
* VS Code
* Git e GitHub

---

## Como Usar
* 1. Baixe o repositório (Clique no Botão “Code” > Download ZIP)
* 2. Extraia a pasta
* 3. Abra o arquivo index.html no navegador

## Acesse com o link abaixo:
* https://nunes-1811.github.io/geradordedicas

- No terminal do VS Code digite:
-	git clone https://github.com/usuario/nome-do-repo.git

* No menu: File(Arquivo) > Clone a repositor(Clonar repositório) > Selecione a aba URL, no campo URL, cole o link do repositório https://github.com/usuario/nome-do-repo.git
